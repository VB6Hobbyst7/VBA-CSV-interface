# fileName
{: .fs-9 }

Gets the file name that corresponds to the last opened CSV file.
{: .fs-6 .fw-300 }

---

## ReadWrite

_ReadOnly_

---

## Syntax

*expression*.`fileName`

### Parameters

_None_

### Returns

*Type*: `String`

---

## Remarks

The `fileName` property is set when CSV file is load to memory.
