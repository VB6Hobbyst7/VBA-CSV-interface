# CSV connection status

## Description
Gets the CSV connection status for the current instance. When False, the CSV file is not open in memory.

## Syntax
*expression*.**Connected**{: .fs-3 .fw-300 }

### Parameters

**None**

### Returns

Type: `Boolean`
