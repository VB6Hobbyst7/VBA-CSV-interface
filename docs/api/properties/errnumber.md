# ErrNumber
{: .fs-9 }

Gets the number for the last occurred error over the current instance.
{: .fs-6 .fw-300 }

---

## ReadWrite

_ReadOnly_

---

## Syntax

*expression*.`ErrNumber`

---

### Parameters

_None_

### Returns

*Type*: `Long`

---

## Remarks

Use the `ErrNumber` property to check if the last requested operation succeed.
