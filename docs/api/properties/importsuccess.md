# ImportSuccess
{: .fs-9 .fw-300 }

Gets the import operation status for the current instance.
{: .fs-6 .fw-300 }

---

## ReadWrite

_ReadOnly_

---

## Syntax

*expression*.`ImportSuccess`

### Parameters

_None_

### Returns

*Type*: `Boolean`

---

>:pencil: **NOTE:**
>
>A `False` value from `ImportSuccess` property indicates the last import operation wouldn't succeeded.
