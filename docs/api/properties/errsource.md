# ErrSource
{: .fs-9 }

Gets the source for the last occurred error over the current instance.
{: .fs-6 .fw-300 }

---

## ReadWrite

_ReadOnly_

---

## Syntax

*expression*.`ErrSource`

### Parameters

_None_

### Returns

*Type*: `String`

---

## Remarks

Use the `ErrSource` property to check if the last requested operation succeed.
