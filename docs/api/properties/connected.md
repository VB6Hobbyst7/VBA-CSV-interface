# Connected
{: .fs-9 }

Gets the CSV connection status for the current instance.
{: .fs-6 .fw-300 }

---

## ReadWrite

_ReadOnly_

---

## Syntax

*expression*.`Connected`

### Parameters

_None_

### Returns

*Type*: `Boolean`

>:pencil: **NOTE:**
>
>A `False` value from `Connected` property indicates the CSV file is not load in memory.
