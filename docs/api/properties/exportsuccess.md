# ExportSuccess
{: .fs-9 }

Gets the export operation status for the current instance.
{: .fs-6 .fw-300 }

---

## ReadWrite

_ReadOnly_

---

## Syntax

*expression*.`ExportSuccess`

### Parameters

_None_

### Returns

*Type*: `Boolean`

>📝**Note:**
>A `False` value from `ExportSuccess` property indicates the last export operation wouldn't succeeded.
